Install

One-line installer (works on every supported OS):

curl -fsSL https://raw.githubusercontent.com/yousafkhamza/aws-patch/main/install.sh | sudo bash

Or install a specific version's package directly (filenames update automatically above):

sudo dpkg -i aws-patch_*.deb   # Debian/Ubuntu
sudo rpm -i aws-patch-*.rpm    # RHEL/Amazon Linux/Rocky/Alma

Why aws-patch

Never touches GRUBNo bootloader automation, ever.
Never removes a kernelRollback capability always preserved.
Never force-rebootsOnly with --reboot or your confirmation.
Predicts rebootsKnow before you patch, not after.
Auto-repairs broken state--broken-fix handles common failure modes.
AL2023-awareHandles point-release boundaries automatically.
